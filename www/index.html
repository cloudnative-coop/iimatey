Aye Aye, Matey!

call on a trusted mate to explore the terminal with you!

matey is a program to start up a session in your terminal that is sharable via a weblink. It is meant as a low-friction way to collaborate with a friend directly in your command line.

in your terminal, type...

matey start

to start up the pair

matey connect

to connect to someone else

matey status

to check the status of the service

Installation

To install matey, you will first install prerequisite tools that matey requires, then run matey's install script.

install prerequisites

Our matey tool depends on tmux, ttyd, wireguard.

If you are on ubuntu/linux:

copy and paste the following into your terminal

apt install tmux,ttyd,wg
If you are on macos:

first, install brew

Then, copy and paste the following into your terminal:

brew install tmux,ttyd,wg

run our install script

After prerequisites are installed, copy and paste the following into your terminal

curl -fsSL https://raw.githubusercontent.com/ii/matey/canon/iimatey-setup.sh | bash

FAQ

Is matey secure?

To a reasonable degree, yes. When you start a session, we generate a random url that would be virtually impossible to guess and would require treme3ndous time and effort to determine through brute force.

The session exists for only as long as you have the terminal window open.

Still, you are sharing your terminal as a site available on the internet, so some precautions are needed:

  • share the link only with a person you trust
  • close your terminal window at the end of the pair, to make sure you are no longer sharing any sessions
